Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma página web desenvolvida com HTML e CSS, focada em oferecer uma ferramenta de tradução de textos entre diferentes idiomas. A proposta é criar um ambiente virtual acessível, onde o usuário possa digitar ou colar um texto e escolher o idioma de destino para obter uma tradução instantânea e precisa.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de livros em uma seção flexível (.livros). Cada livro é destacado em seu próprio contêiner (.livro), mostrando uma imagem da capa, detalhes do livro e um link de compra. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração da coleção de livros.

Minha Biblioteca: Uma Webpage Personalizada

O site tem como objetivo incentivar o cuidado com a natureza em espaços urbanos, fornecendo informações úteis sobre cada planta, como nome científico, necessidade de luz e rega, além de dicas para cultivo. O HTML estrutura o conteúdo em seções: um cabeçalho com o nome do site, uma introdução explicando o propósito da página e uma área principal com cartões individuais para cada planta.

Cada cartão (.planta) exibe uma imagem, informações básicas e um botão para saber mais. O CSS é responsável por estilizar os elementos com uma paleta de cores verdes e terrosas, fontes naturais do Google Fonts e efeitos suaves ao passar o mouse sobre os botões. O layout é responsivo, adaptando-se automaticamente a celulares e tablets com o uso de flexbox e media queries.

O design transmite leveza e acolhimento, alinhado com o tema natural do projeto. A proposta mostra como HTML e CSS podem ser usados para criar um espaço informativo e inspirador sem a necessidade de programação complexa, valorizando a beleza e o cuidado com as plantas.

Minha Biblioteca: Uma Webpage Personalizada

Este projeto é um site desenvolvido com HTML e CSS, ideal para pessoas que buscam preparar refeições rápidas e práticas no dia a dia. A proposta é oferecer uma coleção de receitas com ingredientes simples, tempos de preparo curtos e instruções objetivas. O HTML organiza o site em um cabeçalho com o logotipo e o menu de categorias, uma seção introdutória e uma área principal com cartões de receitas.

Cada receita é apresentada com imagem, tempo de preparo, número de porções e um link para a página completa. O CSS dá vida ao visual com cores vibrantes, ícones personalizados e uma tipografia amigável.

O design responsivo usa flexbox e media queries para se adaptar bem a dispositivos móveis, e os botões ganham destaque com efeitos de transição ao serem clicados. A interface incentiva a navegação rápida e intuitiva, facilitando a escolha de receitas de acordo com o tempo disponível ou os ingredientes em casa. Sem necessidade de JavaScript, o projeto demonstra como HTML e CSS podem criar uma experiência funcional e envolvente, ajudando quem cozinha a encontrar soluções rápidas sem perder o sabor.
